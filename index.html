imageCSS 中文版

做你图片的掌控者

左浮动 / 左对齐

想让你的图片靠左？没问题… 下面教你怎么做到它并且让它看起来非常神奇。

left { 
    float: left; 
    margin: 5px; 
}

居中（水平）


需要让你的图片呆在一个区域的中央（水平）？

center { 
    display: block; 
    margin-left: auto; 
    margin-right: auto;
}

右浮动 / 右对齐

或者想让你的图片靠右？也不是问题… 以下是方法，可以让它看起来不错。

right { 
    float: right; 
    margin: 5px; 
}

对齐顶部

紧靠顶部的文字。

将文字对齐一张图片的顶部。

top {
vertical-align:top;
}

文本居中（水平）

位于图片中部的文字。

将你的文字排列到一张图片的中部（垂直方向）。

middle {
    vertical-align:middle;
    margin: 5px;
}

对齐底部

紧靠底部的文字。

让你的图片对齐一张图片的底部。

bottom {
vertical-align:baseline;
}
